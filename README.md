### Contribution Guideline
```sh
git clone https://github.com/weDevsOfficial/dokan-migrator.git

# Install npm packages.
npm i

# Install composer packages.
composer install

# Dump composer autoload.
composer du -o

# For Hot reloading
yarn dev:hot

# For dev build
yarn dev

# For production build
yarn build
```
